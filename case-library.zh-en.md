# Case Library / 指标归因案例库

## Case 1: DAU down 20% / DAU 下降 20%

Check:
- Logging or dashboard change
- App release and crash rate
- Channel traffic mix
- New vs returning users
- Core funnel: open → feed view → interaction → retention

Output should include signal validation, segmentation, causal hypotheses, and action.

## Case 2: AI answer satisfaction down / AI 回答满意度下降

Check:
- Model version or prompt change
- Retrieval quality / RAG recall
- Latency and timeout
- Query mix changes
- Badcase taxonomy: hallucination, irrelevant answer, refusal, outdated info

## Case 3: CTR up but retention down / 点击率上升但留存下降

Hypothesis:
- Clickbait content improved short-term clicks but hurt long-term trust.
- Recommendation optimized immediate CTR but reduced diversity.
- Need trade-off metrics: dwell time, negative feedback, 7-day retention.
